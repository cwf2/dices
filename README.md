# Digital Initiative for Classics: Epic Speeches

DICES is a database of metadata on direct speech representation in Greek and Latin epic poetry. It contains information about who says what to whom in epics from Homer to Late Antiquity, including over 4500 speeches, 1000 characters, and 50 poems.

- **Project Website**: [dices.mta.ca](http://dices.mta.ca)
- **Database Search**: [db.dices.mta.ca](http://db.dices.mta.ca)
- **Python Client**: [github.com/cwf2/dices-client](https://github.com/cwf2/dices-client)

This repository contains the Django backend that powers the server.
